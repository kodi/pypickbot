.. _plugin-topic:

***********************************
``topic``: Channel topic management
***********************************

.. module:: topic

This module lets bot admins set the channel message of the day in the
channel topic.

.. command::
    !motd message

    Sets the message of the day.

.. command::
    !motd

    Shows the current message of the day.

.. command::
    !motd --

    Empties the message of the day.
